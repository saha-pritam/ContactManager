I'm excited to share that I've been working on a project that closely resembles the functionality of Google Contacts. While there might be a few features missing currently, rest assured that I have plans to incorporate them in future updates.

For the tech enthusiasts out there, the backbone of this project is crafted using Java, with the assistance of Spring Boot for a robust framework. Thymeleaf takes care of the templating on the server side, ensuring seamless HTML rendering. The data is stored and managed using MySQL, while the front-end is a blend of HTML, CSS, and the power of Tailwind CSS for a sleek design. JavaScript, with a sprinkle of jQuery, adds that extra touch of interactivity to the user interface.
